# Gas-Control
Controlador de gasolina para veículos que não possuem um controle.
# Sistema
O sistema consistirá em receber dados enviar ao banco e retornar as informações necessárias como por exemplo:
  ■ Quantidade abastecida
  ■ Tamanho do tanque
  ■ km inicial
  ■ km hora do calculo
